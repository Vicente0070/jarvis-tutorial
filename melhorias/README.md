# Pasta `melhorias`

Esta pasta guarda planos de melhoria e tarefas planejadas para o projeto JARVIS. Use-a para documentar propostas antes de implementar — assim mantemos um histórico e podemos iterar em pequenos passos.

Estrutura sugerida:

- 0000_template_plano.md — modelo para planos de melhoria
- 0001_adicionar_input_texto.md — plano: permitir executar JARVIS por texto (CLI)

Como usar:

1. Crie um novo arquivo numerado seguindo o padrão `000N_descricao.md`.
2. Preencha o template com objetivo, passos e arquivos a modificar.
3. Peça para eu implementar um plano específico — eu seguirei o processo: analisar, propor mudanças, implementar e rodar verificações.

Regras rápidas:

- Um plano por arquivo.
- Numerar sequencialmente (0001, 0002, ...).
- Ao aprovar um plano, indique se posso editar o código e commitar (confirmar ação de Git/push se desejar).
